Fully textured high detailed scale modelism table with scale model during the painting process. The scene contains materials and textures. The 3ds max file also contains vray 1.48.03 (lighting and rendering setup) and scanline version ready to use.

The scene includes all materials and textures needed to render the images above. It includes detailed models you can use in your scenes:

- Airbrush
- Part of Mercedes Class C AMG
- Brushes

These model has been converted from 3ds max to other formats using Deep Exploration 4. Although every models has been checked with the appropriate software, if you have any problem with the format you have downloaded, please try any other one or feel free to contact with me if you need further information.
